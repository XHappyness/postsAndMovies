# postsAndMovies
一个小程序的demo，包含新闻列表页面、新闻详情页、电影列表页、电影排行榜页等


### 技术文档
* [weixin](https://mp.weixin.qq.com/debug/wxadoc/dev/)


### Demo的开发目录结构
* commom 共用的一些方法
* data 页面需要的数据
* images 图片
* pages 各个页面
    * index 欢迎页面
    * movies 电影模块
    * post 新闻模块
* app.* 设置全局的一些信息
* project.config.json 项目配置信息
* 微信小程序demo.wmv 项目的效果演示视频
